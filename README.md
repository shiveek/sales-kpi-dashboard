# Sales KPI Dashboard

## Overview

The Sales KPI Dashboard is an interactive business analytics dashboard built using Streamlit, Pandas, and Plotly. It helps users monitor sales performance, profit trends, order metrics, and product performance through dynamic visualizations and KPI cards.

## Features

* Total Sales KPI
* Total Profit KPI
* Total Orders KPI
* Average Order Value
* Monthly Sales Trend Analysis
* Sales by Category
* Profit by Region
* Top Products Analysis
* Interactive Filters

  * Region Filter
  * Category Filter
* Responsive Dashboard Layout

## Technologies Used

* Python
* Streamlit
* Pandas
* Plotly

## Installation

1. Clone the repository:

git clone https://github.com/your-username/sales-kpi-dashboard.git

2. Navigate to the project folder:

cd sales-kpi-dashboard

3. Install dependencies:

pip install -r requirements.txt

4. Run the application:

python -m streamlit run app.py
